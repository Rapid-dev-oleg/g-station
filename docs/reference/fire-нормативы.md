## 1. Нормативный расчёт расхода и запаса (СП 8.13130.2020)

Высокая уверенность, первоисточник meganorm.ru.

- **Наружное ПТ, населённый пункт — Таблица 1:** расчётное число одновременных
  пожаров 1–5 по населению (≤10 тыс. → 1; 10–25 тыс. → 2; 100–1000 тыс. → 3;
  >1 млн → 5). Расход на один пожар — от 5–10 л/с (малые поселения) до 110 л/с
  (крупные города), по населению + этажности.
- **Наружное ПТ, отдельное здание Ф1–Ф4 — Таблица 2:** кросс-таблица по
  строительному объёму И этажности; от **10 л/с** (≤1000 м³) до **35 л/с**
  (150–200 тыс. м³). Промышленные/складские Ф5 — отдельные Таблицы 3/4.
- **Продолжительность тушения (п. 5.17):** **3 ч**; **2 ч** — для зданий I–II
  степени огнестойкости с негорючими несущими конструкциями и утеплителем.
- **Объём пожарного запаса (п. 9.2):** `V = расчётный расход × продолжительность`.

## 2. Гидравлика и подбор насоса

- **Рабочая точка Q/H:** Q — из ТЗ/гидрорасчёта; требуемый напор `Hсум = Hгео +
  Hостат + ΣHтр + ΣHмест`. Точка ТП — с запасом 5–10 % (методика скила).
- **Спринклер (СП 5.13130):** расход оросителя `q = 10·K·√P` (P в МПа), эквивалент
  `Q = k·√H`. Пример: K=0,42 (RA1325), P=0,12 МПа → q≈1,455 л/с. ⚠️ Это СП 5.13130
  (АУПТ), не СП 10/СП 8.
- **Мощность двигателя:** `P = ρ·g·Q·H / η`, монтажный насос с запасом ×1,15,
  округление вверх до стандартного ряда.
- **Резервирование:** ≥1 рабочий + 1 резервный одинаковой Q/H, **АВР** (резерв
  включается автоматически за 10–30 с). Крупные объекты — 2 раб. + 1 рез.;
  насосы ставят парами (основной+резервный), до 8 шт.

## 3. Состав спецификации (валидировано, 5 производителей)

- **Насосная группа:** основной + резервный насосы (одинаковые Q/H, АВР); жокей —
  многоступенчатый, подача < расхода 1 спринклера, напор +5–10 м; мембранный
  (расширительный) бак с манометром.
- **Обвязка:** всасывающий и напорный коллекторы (AISI 304); **≥2 всас и ≥2
  напор линий** независимо от числа насосов (СП 8 п.7.5/7.6, для категорий I–II);
  на каждый насос — **задвижка на входе и выходе + обратный клапан на напоре +
  манометр/датчик давления перед обратным клапаном** (СП 10) → насос снимается в
  ремонт без остановки станции.
- **КИП:** манометры, **электроконтактный манометр**, реле/датчик давления,
  **реле сухого хода**, **расходомер**, датчики уровня (подземное).
- **Автоматика:** шкаф управления (прямой/плавный пуск, АВР).
- **Корпус/запас:** блок-бокс/павильон/подземная стеклопластиковая ёмкость/рама;
  резервуары пожарного запаса (объём из п.2).
- **Сценарные опции:** компрессор (воздушная dry-pipe АУПТ), **дизель-генератор**
  (автономная работа), **водоподготовка**, дренажный насос (подземное), **узел
  подключения пожарной техники (МПТ)**, **удалённый мониторинг**.

## 4. Смета (структура затрат)

Позиции спецификации → смета 1:1 (cyberleninka): цены насосов + системы
управления + входного/выходного коллекторов + трубопроводной арматуры + рамы +
работы по сборке + наценка поставщика. Вход процесса — **опросный лист**, выход —
**ТКП** (расход → напор → подбор → спецификация → смета).

## 6. Открытые вопросы (уточнить у клиента)

- **Коллектор уже содержит КИП** (замечание клиента 23.07.2026): в продукции
  G-FIRE узел коллектора, вероятно, включает манометры/реле давления → КИП может
  быть НЕ отдельной группой спецификации, а частью коллектора. **Уточнить**,
  прежде чем перестраивать схему спецификации (сейчас КИП — отдельная группа).
- Актуальная редакция норм: протокол — СП 10.13130.2020 / СП 8.13130.2009 /
  НПБ 88-2001*; презентация — СП 8/10/485. Уточнить, что применяем (влияет на
  таблицы расхода/струй).
- Структура документа ТКП/расчётного дела (разделы, правила шифра) — из внешних
  источников не выведена; берём из скила (`процесс-расчёта.md`, шаг5-выход).
