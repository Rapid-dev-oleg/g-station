# Как мы рассчитываем пожарные насосные станции

Презентация для клиента. Расчёт идёт по конвейеру из трёх этапов:
**парсинг → расчёт → подбор**. Нормативная часть («скелет» станции)
считается автоматически и стабильно; экспертные решения (конкретная
модель насоса, производитель) подтверждает инженер.

---

## Этап 1. Парсинг — собираем исходные данные

Загружаем пакет документов клиента «как есть»: ТЗ, опросный лист, проект
(ВК / гидрорасчёт), заметки менеджера, ТКП конкурента.

- Распознаём тип каждого файла и извлекаем текст (docx, pdf, xlsx, txt).
- Определяем тип станции (пожаротушение) и сценарий: подбор с нуля /
  проверка чужого подбора / замена конкурента / торги.
- Собираем **карточку параметров**: Q, H, схема резервирования, жокей,
  исполнение, резервуары, ограничения (бюджет, габариты, сроки).
- Каждое число помечается источником: *взято из документа* / *выведено* /
  **допущение**. Допущения выписываются явно — ничего не выдумывается молча.

→ **Гейт инженера:** проверяет и правит карточку перед расчётом.

---

## Этап 2. Расчёт — определяем «что нужно»

Из карточки вычисляем целевые характеристики станции по нормативам
(СП 8/10/485) и инженерным формулам.

| Что считаем | Как считаем |
|---|---|
| **Рабочая точка Q/H** | `Q` — из ТЗ (л/с → м³/ч ×3,6) или из гидрорасчёта (расход струи × число струй). `H = H требуемый − H на вводе`. Точка ТП — округление вверх с **запасом 5–10 %**. |
| **Запас пожарной воды** | `V = расход × время тушения` (по СП 8/10/485), округление вверх. Задаёт объём резервуаров. |
| **Схема резервирования** | По умолчанию `1/1`; `2/1` — если один насос не покрывает расход. Число насосов = рабочие + резерв (1/1→2, 2/1→3, 2/2→4). |
| **Диаметр коллектора** | По расходу станции (всас 0,8–1,5 м/с, напор 1,0–2,5 м/с). `DN = max(по расходу; патрубок насоса + 2 типоразмера)`. Всас крупнее напора. Возможен разброс ±1 типоразмер. |
| **Мощность двигателя** | `P_вал = Q·H / (367·η)`, η = 0,6–0,75. `P_двиг = P_вал × k` (k от 1,1 до 1,5 по мощности), округление вверх до стандартного ряда. |
| **Тип пуска** | Для пожарных по умолчанию **прямой** (надёжность); плавный — при питании от генератора, слабой сети или ограничении пускового тока по ТУ. |

→ Результат — точные требования к станции, независимо от бренда.

---

## Этап 3. Подбор — определяем «чем закроем»

Под рассчитанные характеристики подбираем конкретное оборудование.

| Что подбираем | Как подбираем |
|---|---|
| **Основной насос** | По рабочей точке Q/H — класс и конструктив; типоразмер и мощность по оценке этапа 2; число по схеме. Сверяем по прайсу, что типоразмер существует. *Точная модель и бренд — за инженером (нужны напорные кривые завода и наличие на складе).* |
| **Жокей-насос** | Если нужен — вертикальный многоступенчатый по Q/H (`H = H осн. + 10 м`) + мембранный бак ≥40 л. |
| **Коллектор** | Диаметр всас/напор — из этапа 2; материал Ст.20 (нержавейка AISI 304 — подземное). Шифр `D-N-d`. |
| **Шкаф управления** | `ШУФ` (прямой пуск) / `ШУФС` (плавный), бренд «Шторм». Номинал — по мощности двигателя; опции: АВР, защиты, уличное/подземное исполнение. |
| **Корпус и резервуары** | Исполнение из карточки; объём резервуаров — из запаса воды (этап 2); материал по исполнению. |
| **Доп. оборудование** | Обратный клапан + затвор на каждый насос, реле, манометры; компрессор — для воздушной АУПТ; для подземного — дренажный насос и датчики уровня. |

→ **Гейт инженера:** подтверждает модель насоса и производителя.

---

## Из чего состоит пожарная станция G-Fire

**Насосная группа**
- Основные насосы (рабочие + резерв по схеме)
- Жокей-насос — компенсатор утечек (ВПВ/АУПТ жилых и общественных зданий)

**Обвязка и коллектор**
- Всасывающий и напорный коллекторы (Ст.20; AISI 304 — подземное)
- На каждый насос: обратный клапан + дисковый затвор
- Патрубки МПТ DN80 с головками — для наружного ПТ (≥2 шт.)

**Автоматика**
- Шкаф управления `ШУФ` / `ШУФС` (бренд «Шторм»)
- Реле сухого хода, пусковые реле, манометры
- Опции: АВР, защиты, уличное/подземное исполнение; либо без ШУ — под чужую
  автоматику («Болид»)

**Корпус и запас воды**
- Исполнение: павильон / блок-бокс / подземная стеклопластиковая ёмкость /
  береговой модуль / моноблок на раме
- Резервуары пожарного запаса (металл — наземное, стеклопластик G-Drum —
  подземное)
- Подземное исполнение: дренажный насос + датчики уровня

**Доп. оборудование по сценарию**
- Компрессор — воздушная спринклерно-дренчерная АУПТ
- Береговая ПНС (забор из водоёма): самовсасывающий + вакуумный насос,
  донный клапан, заборный рукав, ШУ серии `ШУС`

---

## На выходе

**Расчётное дело** — обоснованная спецификация станции и смета. В составе
ТП серии G-Fire:
- характеристики и комплектация,
- **пожарная гидравлическая схема** (реле сухого хода, пусковые реле, манометры),
- габаритный чертёж,
- техлист на насос,
- маркировка изделия,
- патрубки МПТ — для наружного ПТ.

**Итог:** нормативный «скелет» станции система считает автоматически и
стабильно; конкретную модель насоса и производителя подтверждает инженер.
