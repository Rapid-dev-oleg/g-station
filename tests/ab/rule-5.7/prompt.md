# Правило 5.7 — материал коллектора (для LLM-варианта A/B-теста)

Тебе даны характеристики насосной станции пожаротушения. Реши, из какой стали должен быть собран **коллектор станции**: углеродистой Ст.20 (стандарт) или нержавеющей AISI 304 (защита от коррозии).

## Правило

Нержавеющая сталь применяется, если выполнено **хотя бы одно** условие:

1. **HARD-b — ТЗ-нерж.** В техническом задании прямо указано
   `collector_material = "нержавеющая-сталь"`.

2. **HARD-a — Подземное исполнение.** Станция размещена в подземной камере или заглублённой ёмкости — `station_enclosure` ∈ {`подземное-стеклопластик`, `стеклопластиковый-колодец`}, либо `installation_place = "заглублённая"`. Грунтовая влага → коррозия углеродистой стали.

3. **HARD-c — Питьевая среда.** Станция работает на питьевой воде —
   `purpose = "хоз-питьевое"` или `pumping_medium.medium = "питьевая"`.

Если **ни одно** условие не выполнено — углеродистая сталь Ст.20 по ГОСТ 10704-91.

## Формат ответа

Только JSON, без дополнительного текста:

```json
{
  "material": "нержавеющая-сталь" | "углеродистая-сталь",
  "triggerId": "hard-a-underground" | "hard-b-tz-required" | "hard-c-potable" | null,
  "reasoning": "коротко, по какому пункту"
}
```

`triggerId` — id сработавшего условия (или `null`, если использован default).

## Вход

Тебе дадут JSON `StationInput` со следующими (релевантными) полями:
- `station_enclosure`, `installation_place` — для HARD-a
- `collector_material` — для HARD-b
- `purpose`, `pumping_medium.medium` — для HARD-c

Если поле отсутствует — считай его не сработавшим (не `undefined === значение`).
